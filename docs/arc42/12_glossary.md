# 12. Glossary

*   **TTS:** Text-to-Speech synthesis.
*   **SSML:** Speech Synthesis Markup Language.
*   **Hexagonal Architecture:** Architectural pattern emphasizing separation of concerns.
*   **Orchestrator:** Service layer pattern coordinating multiple sub-processes in generation.
*   **Repository:** Infrastructure layer component for data access.
*   **TTFB:** Time to First Byte (latency metric).
*   **CUDA:** NVIDIA platform for parallel computing.
